How to install BuildBot
=======================

1. For windows, follow instructions at http://trac.buildbot.net/wiki/RunningBuildbotOnWindows
   For other OSes, similar steps should be followed

2. Use code/build/buildbot/master.cfg to generate a local buildbot slave
   instructions at http://buildbot.net/buildbot/docs/current/tutorial/firstrun.html (buildslave create-slave)

3. Modify buildbot.*.egg/buildbot/changes/hgpoller.py to enforce UTF-8 comments
   (otherwise it'll choke when non ASCII-characters have been used in the svn history)
